# ISRaD-v2-analysis
The repository contains all the code to reproduce the analysis for the ISRaD v2 publication. 

## Overview of code in the repository:
- ISRaDv2_ClimateZone_AreaCalculation contains all the code to reproduce area calculation for each climate zone (output file saved; no need to re-run script) 
- ISRaDv2_DataDistribution contains all the code to reproduce the Figures 1, 2, S3, and S4
