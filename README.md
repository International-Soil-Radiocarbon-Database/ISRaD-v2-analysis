# ISRaD-v2-analysis
The repository contains all the code to reproduce the analysis for the ISRaD v2 publication. 

## Overview of code in the repository:
- ISRaDv2_Global_SOC_Stocks_Climate_Zones.R contains all the code to reproduce global SOC stock estimates for each climate zone (output file saved; no need to re-run script) 
- ISRaDv2_DataDistribution.R contains all the code to reproduce the Figures 2, 3, S3, and S4
- ISRaDv2_DataDistribution_DataTypes.R contains all the code to reproduce Figure 4 (including stats for data distribution across data types in ISRaD)
- ISRaDv2_DataDistribution_Fractions.R contains all the code to reproduce Figure 5 (including stats for data distribution of fractions in ISRaD)
- ISRaDv2_Wetlands_DataExtraction.R contains all the code to extract the wetland data used in Figure 6 (which was generated using ArcGIS)
- ISRaDv2_NumberStudies_PublicationYear.R contains all the code to reproduce the Figures S1 and S2
